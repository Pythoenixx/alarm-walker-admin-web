# Alarm Walker Admin Web



This repository contains the deployed Flutter Web build for the Alarm Walker admin panel.



The admin panel is used to monitor application issue logs, view reported crashes, and support system debugging during the final year project demonstration.

Render deployment URL:

[https://alarm-walker-admin-web.onrender.com/]

## Notes



This repository contains generated web build files from the Flutter project. Source code changes should be made in the main Alarm Walker repository, then rebuilt and redeployed here.
